# game-
限于体积，上传了部分对象的代码。分别为小怪、boss、摄像头、主角、manager。
游戏在一片黑暗区域，玩家会遇到自动追踪的小怪，可以通过射击击杀。还有大怪，需要通过障碍物躲避。
目的是通过右上角的安全门。
